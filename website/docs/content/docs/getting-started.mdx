# Getting Started with Neo Service Layer

Welcome to the Neo Service Layer documentation! This guide will help you get started with integrating our services into your Neo N3 blockchain applications.

## Overview

Neo Service Layer provides a comprehensive suite of services for building decentralized applications on the Neo N3 blockchain. Our services include:

- Price Feeds
- Contract Automation
- Gas Bank
- Function Management
- Secrets Management
- API Services

## Quick Start

### 1. Connect Your Wallet

First, connect your NeoLine wallet to access our services:

```javascript
import { useWallet } from '@neo-service-layer/react';

function App() {
  const { connect, address } = useWallet();

  return (
    <button onClick={connect}>
      {address ? `Connected: ${address}` : 'Connect Wallet'}
    </button>
  );
}
```

### 2. Access Price Feeds

Get real-time price data from our oracle network:

```javascript
import { usePriceFeed } from '@neo-service-layer/react';

function PriceDisplay() {
  const { price, loading } = usePriceFeed('NEO/USD');

  if (loading) return <div>Loading...</div>;
  return <div>NEO Price: ${price}</div>;
}
```

### 3. Set Up Contract Automation

Create automated tasks for your smart contracts:

```javascript
import { AutomationService } from '@neo-service-layer/core';

const automation = new AutomationService();

await automation.createTask({
  name: 'Daily Distribution',
  contract: 'NeoTokenContract',
  method: 'distribute',
  schedule: '0 0 * * *', // Daily at midnight
});
```
