## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (see `.env.example`)
4. Run examples: `npm run examples`

## Prerequisites
- Node.js 16+
- Neo N3 Wallet
- Access to Neo N3 network
- API keys (if required)
